OSX_cleanup.sh

This script is used to perform maintenance tasks on a Mac. The tasks that are performed include:

Checking for updates to macOS
Emptying the trash can
Clearing the system cache
Clearing the browser cache
Checking the disk for errors
Running App Store updates
Running a disk cleanup utility
Optimizing the startup disk
Running a malware scan

To use this script, first make sure that you have the following packages installed:
softwareupdate
mas
dfcleanmgr
malwarebytes

Then, copy the script to a directory where you want to run it. Finally, run the script by typing the following command in a Terminal window:

bash OSX_cleanup.sh
